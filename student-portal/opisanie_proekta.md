# Описание проекта (как для человека)

## Что вообще было сделано?
Изначально был просто **HTML/CSS/JS** макет на React (просто внешний вид), где ничего не сохранялось. Если перезагрузить страницу — всё пропадало.
Моя задача была превратить этот "макет" в полноценное веб-приложение.

## 1. База данных (Supabase)
Чтобы данные (оценки, домашка) где-то жили реально, я подключил **Supabase**. Это облачная база данных (PostgreSQL).
*   Теперь, когда ты добавляешь "Сделать математику", это улетает на сервер и сохраняется там.
*   Можно зайти с телефона, с компа — данные везде одинаковые.

## 2. Авторизация и безопасность
Самое важное, что я добавил — это **Личный кабинет**.
*   Раньше сайт был один на всех (как доска объявлений).
*   Я сделал систему **Входа и Регистрации** (через Email/Пароль).
*   На уровне базы данных настроил защиту (RLS - Row Level Security).
    *   Это значит: если Вася заходит, он видит *только* свои оценки.
    *   Если Петя заходит — он видит свои.
    *   Никто не может подсмотреть чужие данные, даже если попытается взломать запрос. База сама фильтрует: "ты кто? ага, вот тебе только твои записи".

## 3. Деплой (Хостинг)
*   Чтобы сайт был доступен в интернете 24/7, я залил его на **Surge.sh**.
*   Настроил переадресацию (SPA Routing), чтобы ссылки типа `/homework` или `/grades` открывались правильно, а не выдавали ошибку 404.

## 4. Мобильная версия и Казахский язык
*   Макет был под компьютер. Я адаптировал его под мобилки: добавил **Бургер-меню** (три полоски), которое красиво выезжает сбоку.
*   Перевел весь интерфейс на казахский язык: дни недели, статусы заданий, кнопки.

## Итог
Был просто "рисунок" сайта, стало **SAAS-приложение** (Software as a Service) с базой данных, авторизацией и мобильной версией.
Стек технологий: **React + TypeScript + Vite + Supabase + TailwindCSS**.
Ссылка: [school-helper-ak1ra.surge.sh](http://school-helper-ak1ra.surge.sh)
